# Blinkit_SQL_Project

## Project Overview

This project focuses on analyzing Blinkit (online grocery delivery) data using SQL to extract meaningful business insights. The goal is to understand customer behavior, sales performance, product trends, and operational efficiency through structured queries.

The project is designed to showcase SQL skills such as data cleaning, joins, aggregations, subqueries, CTEs, and window functions, making it suitable for data analyst / data science portfolios.

## Objectives

• Analyze total sales and revenue trends.
• Identify top-selling products and categories.
• Understand customer purchasing behavior
• Analyze order frequency and average order value
• Evaluate delivery performance and ratings
• Perform data cleaning and transformation using SQL

## Tools & Technologies

• Database: MySQL
• Language: SQL
• IDE: MySQL Workbench
• Version Control: Git & GitHub

## Insights Generated

• Identified high-revenue categories contributing most to sales
• Found top-performing products based on order count
• Analyzed customer order patterns and repeat purchases
• Evaluated delivery efficiency and customer ratings

## Conclusion

This Blinkit SQL Data Analysis Project demonstrates how raw transactional data can be transformed into meaningful business insights using structured query language (SQL). Through effective data cleaning, joins, aggregations, and advanced querying techniques, the project uncovers patterns in sales, customer behavior, and product performance.
